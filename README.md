# 1440 Planner

> Every day is 1440 minutes. Plan yours.

A productivity calendar that operates in minute-time (0–1440) instead of
conventional clock hours. Features a paired watch face for Android Wear OS
and Apple Watch.

## Quick Start

```bash
npm install
npx expo start
```
